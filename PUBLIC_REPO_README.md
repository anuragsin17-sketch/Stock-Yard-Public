# Stock Yard - Live Stock Screening Dashboard

🔴 **Live Dashboard:** https://anuragsin17-sketch.github.io/Stock-Yard-Public/

## About

This is the public-facing dashboard for Stock Yard automated stock screening system. The dashboard displays real-time screening results for NIFTY 500 stocks across multiple technical analysis patterns.

## Features

- ✨ **Golden Stocks**: Fibonacci + Trendline + Vertical Line analysis
- 📈 **Volume Breakout**: Enhanced volume spike detection with retracement
- 📊 **W-Pattern**: Weekly double bottom formations
- 🌊 **Elliott Wave**: Long-term Golden Pocket setups
- 📦 **Darvas Box**: Multi-year consolidation breakouts

## Updates

The dashboard is automatically updated **3 times daily**:
- 10:00 AM IST
- 02:00 PM IST
- 08:00 PM IST

## Technology

- **Frontend**: Pure HTML/CSS/JavaScript with Tailwind CSS
- **Data**: JSON-based real-time updates
- **Hosting**: GitHub Pages
- **Mobile**: Optimized for iOS/Android WebView

## Data Structure

The `data.json` file contains:
- Screening results for all patterns
- Stock details (price, 52-week high/low, etc.)
- Diagnostics (stocks processed, matches found)
- Timestamp of last update

## Note

This repository contains only the public dashboard. The screening algorithms and backend code are maintained in a private repository for proprietary protection.
